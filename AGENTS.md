# AGENTS.md — 정몽주 AR MVP

## 프로젝트
용인 문화유산 AR 실증 MVP. 단일 HTML 프로토타입을 확장해 비주얼 퀘스트·백엔드·실증로그·동적QR 추가. 상세는 TASK.md.

## 작업 규칙 (실패 사례 기반)
- 기존 구현(소환·대화·TTS·지오펜스)을 깨지 말 것. 수정 후 반드시 회귀 확인.
- API 키를 클라이언트에 두지 말 것. P3부터 서버 환경변수로만.
- AI 비전을 모든 사진에 호출하지 말 것. 자유 미션 한정 + 호출 상한 + 캐싱.
- iOS에서 WebXR 지면배치에 의존하지 말 것. 이미지 타겟 우선.
- AR 실패가 앱 전체를 죽이지 않게 할 것. 채팅+미션 폴백 유지.
- 개인식별정보(이름·연락처) 저장 금지. 익명 세션ID만.
- 미확정 수치(좌표·반경·시간·상한)를 임의 확정하지 말 것. README에 [CONFIRM] 표시 후 기본값 진행.
- 과설계 금지. 로그 저장은 JSON/SQLite 수준. 불필요한 프레임워크 추가 금지.

## 검증
- 각 기능 추가 후: 데모 모드(키 없이)에서 전체 흐름이 끊김 없이 도는지 먼저 확인.
- 미션은 인증 통과 없이 다음으로 넘어가면 안 됨.
- 키 노출 여부: 빌드 산출물·네트워크 응답에 키 문자열이 없는지 확인.

## 진행 순서
P1·P2(데모) → P3(서버·키이전) → P4(로그) → P5(동적QR). 단계마다 회귀 확인.
